# Gridfinity OpenSCAD

Inspired by https://github.com/vector76/gridfinity_openscad

OpenSCAD scripts for creating gridfinity compatible components

Wifi SVG Command
```
qrencode -t SVG -o wifi.svg "WIFI:S:MyWifi;T:WPA;P:MyPassword;;"
```
will need to remove `<rect fill="#ffffff"/>` to render correctly.
